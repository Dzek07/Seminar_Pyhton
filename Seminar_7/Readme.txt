Описание работы программы

Модуль main - служит для запуска программы

Модуль menu - в этом модуле находится цикл который воспроизводит меню и вызывает необходимые модули. 
В этот модуль импортированны все модуля кроме main 

Модуль calculations - Этот модуль отвечает за расчеты. В нем две основные функции. Функция calc считает 
обычные расчеты реалезованный без функции eval. Функция calc_compl считает комплексный числа реалезованный через
функцию eval. В него импортирован модуль logging для отправки результатов в файл с логами.

Модуль input_user - Этот модуль отвечает за ввод данных от пользователя. В него импортирован модуль logging
для логирования всех вводимых данных.

Модуль print_data - отвечет за вывод всех данных на экран пользователя

Модуль logging - отвечает за запись всех действий программы в файл log.txt и чтание из файла с выводом на экран по запросу пользователя

Файл log.txt - в нем хранятся логи программы
